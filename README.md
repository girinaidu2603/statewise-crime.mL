# State-wise Crime Analysis using Machine Learning

This project analyzes state-wise crime data collected from NCRB reports
using Machine Learning and Data Visualization techniques.

## Technologies Used
- Python
- Flask
- Scikit-learn
- React.js
- GitHub

## Features
- Crime pattern analysis
- Crime prediction using ML
- Interactive dashboard (frontend)
- Modular backend architecture

## Objective
To support public safety planning using data-driven insights.
